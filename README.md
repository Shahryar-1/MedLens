# MedLens

MedLens is an AI-powered medical assistant designed to simplify medicine management using **Computer Vision**, **OCR**, and **Artificial Intelligence**.

The goal of this project is to help users identify medicines, organize their medications, and receive reliable medicine-related information through an intuitive and accessible application.

---

## Current Progress

The following features have been implemented:

- ✅ Project architecture and modular codebase
- ✅ Webcam integration using OpenCV
- ✅ Live camera preview
- ✅ Image capture and storage
- ✅ Image preprocessing pipeline
  - Grayscale conversion
  - Gaussian Blur
  - Binary Thresholding
- ✅ Basic image processing utilities
- ✅ Camera testing framework

---
